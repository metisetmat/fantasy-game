# Interaction MVP #3 — Transition offensive

---

# 1. Objectif tactique

La transition offensive représente :

> la tentative immédiate d’exploiter un déséquilibre adverse après récupération du ballon.

Et honnêtement :
dans ton sport,

# c’est probablement :

l’interaction la plus importante du moteur.

Pourquoi ?

Parce que :

* peu de joueurs ;
* grands espaces ;
* jeu pied/main ;
* turnovers dangereux ;
* gardien volant ;
* pression haute.

Donc :

# chaque perte peut devenir létale.

Très important.

---

# 2. Philosophie moteur

La transition offensive :

* est rapide ;
* opportuniste ;
* contextuelle ;
* fortement émotionnelle.

Elle oppose :

# vitesse de projection

VS

# vitesse de réorganisation adverse.

Très important.

---

# 4. Déclencheur principal

La transition offensive démarre lorsque :

```text id="0hz1wv"
changement possession
ET
bloc adverse partiellement désorganisé
```

Très important.

---

# 5. Fenêtre de transition

Concept clé.

Une transition :

# possède une durée limitée.

Exemple :

```text id="u8c6p9"
2 à 6 ticks
```

Pendant cette fenêtre :

* les espaces sont plus grands ;
* la structure adverse est fragile ;
* la variance augmente.

Puis :
la défense se reforme progressivement.

Très important.

---

# 6. Philosophie décisionnelle

Le moteur doit d’abord décider :

# L’équipe attaque-t-elle immédiatement ?

OU

# recycle-t-elle ?

Très important.

```text id="6lhcgl"

```

* fatigue ;

* variance.

Très important.

# 20. États possibles transition

## Transition explosive

Défense totalement ouverte.

Très dangereux.

---

## Transition contrôlée

Attaque progresse mais défense survit.

---

## Transition étouffée

Défense ralentit suffisamment.

Possession stabilisée.

---

## Turnover transitionnel

Chaos immédiat.

Très important :
possible transition inverse.

# 25. Ce que cette interaction doit révéler

Elle doit permettre d’identifier :

* les équipes létales ;
* les équipes vulnérables après perte ;
* les équipes qui contrôlent le chaos ;
* les équipes qui paniquent ;
* les différences de verticalité.

---

# 26. Critères de succès spécifiques

## Succès si :

* les transitions semblent dangereuses ;
* les espaces apparaissent naturellement ;
* les replis sont visibles ;
* les styles influencent fortement les choix ;
* les turnovers deviennent émotionnels.

---

# Échec si :

* toutes les transitions marquent ;
* aucune transition n’est dangereuse ;
* les équipes se reforment instantanément ;
* les espaces n’existent jamais ;
* le chaos devient illisible.
