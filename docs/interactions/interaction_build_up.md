# Interaction MVP #2 — Construction offensive

---

# 1. Objectif tactique

La construction offensive représente :

> la capacité d’une équipe à faire progresser une possession organisée contre un bloc défensif relativement stabilisé.

Contrairement à :

* la transition ;
* la sortie de pressing ;

la construction offensive :

# privilégie :

* le contrôle ;
* la manipulation ;
* la patience ;
* la structure.

Très important.

---

# 2. Philosophie moteur

La construction offensive :

* n’est PAS une simple succession de passes ;
* n’est PAS de la possession stérile automatique.

Elle représente :

# une tentative collective :

de :

* déplacer le bloc adverse ;
* créer des décalages ;
* ouvrir des lignes ;
* manipuler les densités.

Très football moderne.
Très important dans ton sport.

---

# 12. Philosophie spatiale

Très important.

La construction offensive :

# est principalement une bataille :

* de largeur ;
* de densité ;
* de déplacements.

---

# Attaque cherche à :

* étirer le bloc ;
* occuper les 5 couloirs ;
* créer weak side ;
* ouvrir demi-espaces ;
* attirer pression d’un côté.

---

# Défense cherche à :

* rester compacte ;
* protéger axe ;
* fermer diagonales ;
* empêcher fixation.

---

# 13. États spatiaux clés

## Bloc compact réussi

Très peu d’espaces.
Construction difficile.

---

## Bloc étiré

Weak side dangereux.

---

## Bloc cassé

Transition vers :

* attaque verticale ;
* duel spatial ;
* finition.

Très important.

---

# 17. Résultats possibles

## Succès majeur offensif

Le bloc :

* est étiré ;
* perd densité ;
* ouvre ligne dangereuse.

Transition :
→ attaque verticale.

---

## Succès mineur offensif

* progression territoriale ;
* maintien pression ;
* fatigue défensive.

---

## Équilibre

Possession continue.

---

## Succès défensif mineur

* ralentissement ;
* circulation stérile ;
* recul offensif.

---

## Succès défensif majeur

* interception ;
* piège pressing ;
* turnover dangereux.

---

# 18. Effets fatigue

## Offensive

Fatigue :

* plus mentale ;
* plus structurelle ;
* liée circulation constante.

---

## Défensive

Fatigue :

* coulissements ;
* concentration ;
* répétition replacements.

Très important :
un bloc compact longtemps sous pression doit finir :

* par craquer ;
  OU
* par subir fatigue mentale.

---

# 19. Impact tempo

→ ralentit tempo.

---

## Construction qui étire bloc

→ prépare accélération brutale.

---

## Construction stérile

→ frustration ;
→ risque turnover futur.

---

## Construction stérile

> “La possession tourne sans réelle progression. Les Ravens ne trouvent pas d’ouverture.”

---

# 22. Ce que cette interaction doit révéler

Elle doit permettre de voir :

* les équipes patientes ;
* les équipes organisées ;
* les équipes capables de contrôler un match ;
* les équipes vulnérables spatialement ;
* les différences de structure.

---

# 23. Critères de succès spécifiques

## Succès si :

* les blocs compacts sont visibles ;
* les équipes de possession semblent différentes ;
* les weak sides émergent naturellement ;
* les temps forts existent ;
* les longues possessions racontent quelque chose.

---

# Échec si :

* la possession ne sert à rien ;
* les blocs se cassent instantanément ;
* les équipes ne contrôlent jamais le rythme ;
* les constructions semblent aléatoires ;
* la largeur n’a aucun impact.
