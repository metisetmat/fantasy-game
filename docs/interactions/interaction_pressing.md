# Interaction MVP #4 — Pressing coordonné

---

# 1. Objectif tactique

Le pressing coordonné représente :

> la tentative collective d’une équipe de réduire le temps et l’espace adverse afin de provoquer :

* une erreur ;
* un turnover ;
* un dégagement ;
* une mauvaise décision.

Très important :
dans ton moteur,

# le pressing n’est PAS :

“courir vers le ballon”.

C’est :

# une organisation collective.

Très football moderne.
Très hockey moderne aussi.

---

# 2. Philosophie moteur

Un pressing réussi :

* enferme ;
* oriente ;
* coupe les soutiens ;
* réduit les options ;
* piège dans la densité.

Un pressing raté :

* ouvre des espaces immenses ;
* casse le bloc ;
* expose la profondeur.

Très important.

---

# 4. Déclencheur moteur

Le pressing coordonné démarre lorsque :

```text id="0d3d9t"
équipe défensive décide :
pression active
ET
structure suffisamment compacte
```

Très important.

---

# 5. Philosophie spatiale

Le pressing :

# cherche avant tout :

à contrôler l’espace.

Pas uniquement :
à récupérer immédiatement.

Très important.

---

Très important.

---

# 19. Résultats possibles

## Pressing réussi majeur

* turnover haut ;
* ballon dangereux ;
* défense désorganisée.

Très dangereux.

---

## Pressing réussi mineur

* recul adverse ;
* dégagement forcé ;
* ralentissement.

---

## Équilibre

Pression continue.

---

## Pressing cassé

Bloc éliminé.
Transition dangereuse.

---

## Pressing catastrophique

* weak side totalement ouvert ;
* surnombre ;
* attaque quasi létale.

Très important.

---

* changements direction ;
* coordination mentale ;

---

## Offensive

* mentalement ;

* physiquement.

Très important.

# 24. Ce que cette interaction doit révéler

Elle doit permettre d’identifier :

* les équipes organisées ;
* les équipes agressives ;
* les équipes disciplinées ;
* les équipes naïves ;
* les équipes capables de résister sous pression.

---

# 25. Critères de succès spécifiques

## Succès si :

* le pressing semble collectif ;
* les pièges spatiaux existent ;
* les équipes compactes sont fortes ;
* les pressings ratés sont punis ;
* la fatigue impacte réellement le pressing.

---

# Échec si :

* le pressing est juste une stat ;
* les blocs ne se cassent jamais ;
* le pressing ne fatigue pas ;
* aucune récupération haute ;
* aucune punition spatiale.
