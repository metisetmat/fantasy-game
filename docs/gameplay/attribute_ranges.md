# Attribute Ranges

All visible and derived player attributes use a 0-100 scale.

| Range | Interpretation |
|---:|---|
| 0-20 | Catastrophic |
| 20-40 | Weak |
| 40-60 | Average |
| 60-75 | Strong |
| 75-90 | Elite |
| 90-97 | World-class |
| 98-100 | Generational |

Ratings of 97+ should be extremely rare. They represent exceptional benchmark players, not normal roster-building targets.

Derived attributes must always be clamped to this 0-100 scale.
