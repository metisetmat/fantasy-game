# Sprint 2P Share Pack

This compact review pack uses standalone generated docs plus bundled source files.

Start with:
- `00-share-manifest.txt` for copied files, bundle contents, commands, and git status.
- `canonical-matchreport-evidence-contract.md` for the Sprint 2P rationale.
- `validation.canonical-matchreport-evidence-contract.md` for the typed evidence and warning checklist.
- `bundle__contracts.md` for MatchReport, evidence, warning, and contract guard definitions.
- `bundle__simulation.md` for canonical evidence builders, warnings, key moments, focus, runFullMatch, and harness diagnostics.
- `bundle__reports.md` for the HTML renderer, copy quality utility, coach-facing copy helper, and share-pack validation.
- `coach-report.latest.html` for a visual full-match harness sample when present.
